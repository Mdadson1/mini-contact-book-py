# Hash Table Implementation in Python

A custom hash table implementation built from scratch in Python using object-oriented programming principles.

## Overview

This project recreates the core functionality of a hash table without using Python's built-in dictionary as the underlying storage structure.

The implementation supports:

* Insertion
* Search
* Deletion
* Collision handling
* Dynamic storage management

## Data Structures

### HashTable Class

The HashTable class manages:

* Key-value storage
* Hash function computation
* Collision resolution
* Table operations

## Concepts Demonstrated

* Object-Oriented Programming
* Classes and Objects
* Encapsulation
* Hash Functions
* Open Addressing
* Collision Resolution
* Algorithm Analysis

## Learning Outcomes

This project was completed to strengthen understanding of:

* How hash tables work internally
* Dictionary-like data structures
* OOP design in Python
* Time complexity of lookup operations
